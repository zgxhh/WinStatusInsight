# WinStatusInsight 产品说明

WinStatusInsight 是一个只在本机运行的 Windows 状态分析工具。它通过本地 Express 服务调用 PowerShell 采集系统状态，再由 Vue/Electron 桌面端展示分析结果。

## 核心目标

- 快速判断电脑卡顿来自 CPU、内存、本地开发服务、浏览器或资源管理器。
- 把多进程应用聚合成更容易理解的应用视角。
- 识别本地开发项目并提供安全的停止入口。
- 保存历史快照，方便对比两次读取之间的压力变化。

## 安全边界

- 默认不上传任何系统数据。
- 结束进程前需要用户二次确认。
- 当前 WinStatusInsight 面板会被标记为受保护，避免误停。
